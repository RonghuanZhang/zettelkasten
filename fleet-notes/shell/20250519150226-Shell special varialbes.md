| 变量           | 说明                  |
| ------------ | ------------------- |
| `$#`         |  命令行参数的个数           |
| `$0`         | 当前脚本的文件名（比如 `hive`） |
| `$1`、`$2`... | 第 1、2 个参数           |
| `$@`         | 所有参数，保留每个参数的边界（推荐）  |
| `$*`         | 所有参数，作为一个整体字符串（不推荐） |
| `"$@"`       | 所有参数，每个作为单独的字符串传递   |
| `"$*"`       | 所有参数，合并为一个字符串       |
