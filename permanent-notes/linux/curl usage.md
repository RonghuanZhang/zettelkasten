## 常用选项

| 选项   | 含义                          | 长选项             |
| ---- | --------------------------- | --------------- |
| `-L` | 如果遇到重定向会继续跟随重定向进行请求         | `--location`    |
| `-O` | 将服务器响应保存成文件，并用URL的最后部分作为文件名 | `--remote-name` |
|      |                             |                 |
